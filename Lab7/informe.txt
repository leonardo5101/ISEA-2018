10. Observaciones y Conclusiones

	- Odoo permite trabajar con varias bases de datos en simultaneo.
	- Odoo permite modificar la información de la compañia, desde el logo hasta información de su ubicación, asi como añadir otra compañia.
	- El módulo de gestión de inventario permite organizar los productos, configurar los almacenes, realizar ajustes de inventario cada cierto tiempo asi como establecer reglas de abastecimiento.
	- Al instalar el módulo de inventario, también se añade el módulo de Debates con el cual se pueden realizar charlas entre los integrantes de la compañia así como reuniones privadas.
	- El módulo de Punto de Venta integra un sistema con interfaz para realizar ventas de los productos existentes de manera eficiente. Permite configurar entre otras cosas, los métodos de pago aceptados, asi como establecer si el Punto de Venta pertenece a un bar / restaurante para activar la organización de mesas.
	- El punto de venta trabaja de manera online y offline, asi que una vez que el servicio se reestableca, se actualizaran los cambios realizados durante el estado offline.
	- El módulo de Compras permite realizar ordenes de abastecimiento a los proveedores registrados. Permite procesar las facturas de estas ordenes y llevar un seguimiento de las facturas pagadas en su totalidad y las que se mantiene una deuda.
